# üè∏ Badminton Training App
A full-stack web application for managing badminton training, allowing instructors to assign drills and track student progress.

## Features
- Role-based access for instructors and students
- Instructors can view and assign training drills
- Students can view assigned drills and mark them as completed
- Progress tracking per student

## Tech Stack
- **Frontend:** React, TypeScript, Material UI, TanStack Query (React Query)
- **Backend:** Django, Django REST Framework

## Purpose
Built as a learning-focused project to practice full-stack development, REST API design, and role-based permissions using a real-world training workflow.
